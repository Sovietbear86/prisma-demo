# Prisma Demo (Next.js + Prisma + Postgres)

CRUD для пользователей и постов на Next.js (App Router) + Prisma.  
База данных: PostgreSQL.

## Requirements
- Node.js (рекомендуется LTS)
- Docker + Docker Compose
- (Windows) рекомендуется запускать проект в WSL
